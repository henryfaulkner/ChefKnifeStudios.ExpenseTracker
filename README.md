# AI Expense Tracker

An AI-powered mobile app for tracking expense in a mindful, efficient manner.

Track your expenses with:
- Monthly budgets
- Receipt scanning
- Speech recognition
- RAG Search
- Excel exports
- Recurring expenses

  ## Built as a personal project to practice topics that I explored at Microsoft Build 2025.

  - .NET MAUI
  - RAG Search
  - Semantic Kernal
